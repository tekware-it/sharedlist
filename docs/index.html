Shared List – Privacy Policy

Last updated: 2026-01-08

1. Introduction

Shared List respects user privacy. This Privacy Policy explains how data is handled when using the app.

Shared List is an open source project. Source code is available on GitHub: https://github.com/tekware-it/sharedlist

2. Data We Collect

The app does not collect personal information such as name, email address, or advertising identifiers.

  • User-generated content (e.g. shared lists)
  • Push notification tokens required to deliver notifications (APNs on iOS, FCM on Android)
  • Installation identifier used by the notification service on Android to route messages (used only for delivery, not for tracking)

3. Data Security

User-generated content is encrypted on the user’s device before being transmitted. The backend only processes and stores encrypted data and cannot access the plaintext.

All network communication uses secure connections (HTTPS).

4. How Data Is Used

  • Provide core app functionality
  • Synchronize shared lists between users
  • Deliver push notifications related to list updates

5. Push Notifications

Push notifications are used only to notify users about shared list updates. They do not contain sensitive or encrypted user content and are not used for marketing or tracking.

iOS: notifications are delivered via Apple Push Notification service (APNs).

Android: notifications are delivered via Firebase Cloud Messaging (FCM) to reach devices reliably. FCM uses an installation identifier and device token to route messages; this is used only for notification delivery.

6. Data Sharing

Data is not sold or shared with third parties for advertising or tracking purposes. Encrypted data is processed only by backend infrastructure required to operate the service.

7. Donations

The app includes optional links to external donation platforms (e.g. Buy Me a Coffee). Donations are voluntary and do not unlock features or content within the app.

8. Data Retention

Encrypted user-generated content is stored only as long as necessary to provide the service and can be removed upon request. Push tokens may be stored to enable notifications and can be removed upon request.

9. Your Rights

You may request deletion of your data by contacting support.
